About Deck Foundry

Effective as of 2026

Vision

This web app has been inspired by the desire to create trading card games. This is meant to be a platform on which users can express creativity and design their own trading card game ecosystem.

The long term goal is for this to be flexible and feature-rich platform on which creators can make a trading card universe as complex or straight-forward as they wish. Discord is great place for a trading card system to reside, and I believe we can provide much more than other trading card bots out there.

How it works

It all starts at deck creation, then creators add cards one-by-one.

  • Create and edit the card template(s) for your deck. Define attributes, rarity, name, card image, etc.
  • Create merge perks, card missions, and set card drop rates.
  • Publish your deck to the public marketplace for other users to adopt and use in their server.
  • All servers using the same trading card deck share the same ecosystem for all users. This means global intentory and currency between Discord servers.
  • Notifications regarding a TCG deck are sent to a user's direct message inbox.

Deck Foundry is an Indie project

So far, this project has been developed by myself by using a vibe coding app. I have a general knowledge of coding so there is some that I can do myself, but there is much of it that is above my head. Currently, this project is financially supported by myself and a friend.

Monetization Philosophy

Monetization is planned, however I do not want to employ predatory tactics when it comes to microtransactions. I want Deck Foundry to have long-term sustainability without taking users for granted. Support from users is greatly appreciated. The least I can do is be fair, transparent, and put Deck Foundry users first. I also would like users to benefit from their creativity. I'm exploring ways for users to monetize what they build on Deck Foundry without creating unfair advantages for players.

Development Status and Future Plans

I continue to develop this with my free time. My free time fluctuates, and I do not have a set development schedule but I wish to continue at a reasonable rate that will keep users engaged.

The plan for Deck Foundry at this time is to continue adding features and improvements on a case-by-case basis, determined by priority. Major bugs and issues are first priority, new features are second priority, and minor issues are third priority.

Planned features:

  • Advanced PvP system
  • Tournament system
  • Limited events system
  • Card image template editor for deck creators
  • Deck editor improvements
  • Deck marketplace enhancements
