How template changes affect existing cards:
  • Add attribute: All existing cards receive a default value — 0 for numbers, empty for text, blank for dropdowns.
  • Remove attribute: The attribute and all its values are permanently deleted from every card.
  • Rename attribute: The name updates everywhere — existing card values are preserved.
  • Remove dropdown option: Cards using that option will have the value cleared to blank.
  • After changes, go to Overview & Cards to update card values as needed.
